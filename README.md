# Apple Pips for Rainbow

A color scheme inspired by the GiveUpArt's artwork for the [Apple Pips][1] record label.

![Screenshot][2]

## Installation

1. Setup [Rainbow][3]
2. Download a `.css` file of choice
2. Include the file in your HTML
